# Boss Bath Sample Site

Static sample landing page for walk-in shower installation leads.

## Preview Locally

Open `index.html` in a browser.

## Share A Link

This site can be hosted as-is on GitHub Pages, Netlify Drop, Cloudflare Pages, or any static web host. No build step is required.

## Form Note

The estimate form currently shows an on-page demo confirmation. Connect `#estimate-form` in `script.js` to a CRM, form backend, or hosted form service when you are ready to capture real leads.
